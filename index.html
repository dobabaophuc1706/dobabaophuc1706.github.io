Questions About My Projects

How to use Control Center Wallpaper?

Step 1: Download your favorite image

Step 2: Install Misaka

Step 3: Install CC WallPaper from misaka.app

Step 4: Hold and tap Advanced Setting then import image downloaded in Step 1 at WALLPAPER IMAGE PATH

Step 5: Edit POSITION and WALLPAPER SIZE to suit your screen resolution (Also remember to change the location of Shazam Module in IOS Setting to match the wallpaper)

Step 6: Apply then Respring (Should be Restart Frontboard)

~ NOTE ~

Add Shazam Module before using

Thank you @straight-tamago for a amazing tweak 🥳

How to apply LS UI (Assets.car) from my Github?

Step 1: Download your favorite style (Assets.car for your IOS)

Step 2: Install Misaka

Step 3: Install LS UI Manager IOS 15 or LS UI Manager IOS 16 from PhucDo's Repository

Step 4: Hold and tap Advanced Setting, then import Assets.car file downloaded in Step 1

Step 5: Apply then Respring (Should be Restart Frontboard)

~ NOTE ~

You also can use different tools such as: Cowabunga, or FileSwitcherPro (which support MDC or KFD overwrite)

How to apply Clock Font from my Github?

Step 1: Download your favorite Font Clock

Step 2: Install Misaka

Step 3: Install Font Manager from EPOSbox

Step 4: Hold and tap Advanced Setting, at CLOCK FONT, import Font file downloaded in Step 1 at iOS 16・ADTTime.ttc (if IOS 16) or iOS 15・SFUISoft.ttc (if ios 15)

Step 5: Edit POSITION and WALLPAPER SIZE to suit your screen resolution (Also remember to change the location of Shazam Module in IOS Setting to match the wallpaper)

Step 6: Apply then Respring (Should be Restart Frontboard)

~ NOTE ~

You also can your different Tools that support Font OverWrite for MDC or KFD such as: Cowabunga (MDC); WDBFont (MDC); KFDFont (KFD);...

Thank you EPOS and Tyler for a amazing tweak 🥳
